# Data Source

The raw CSV file is **not included** in this repository due to file size.

## How to Download

1. Go to the CMS Medicaid Spending by Drug page:
   https://data.cms.gov/summary-statistics-on-use-and-payments/medicare-medicaid-spending-by-drug/medicaid-spending-by-drug

2. Download the file:
   **DSD_MCD_RY25_P06_V20_D23_BGM.csv**

3. Place it in this `/data/` folder before running the SQL load script.
